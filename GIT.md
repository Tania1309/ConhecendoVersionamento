# Git

## O que é git ?
O Git é um sistema de controle de versão distribuído , gratuito e de código aberto, projetado para lidar com tudo, desde projetos pequenos até projetos muito grandes, com rapidez e eficiência.

O Git é extremamente rápido e possui um enorme ecossistema de interfaces gráficas , serviços de hospedagem e ferramentas de linha de comando .

## Como surgiu o git ?
O Git surgiu em 2005 após a comunidade do Linux perder acesso ao BitKeeper, uma ferramenta de versionamento que antes era gratuita. Com isso, Linus Torvalds decidiu criar um novo sistema baseado no que havia aprendido usando o BitKeeper.

O Git foi projetado para:

- Ser muito rápido  
- Ter um funcionamento simples  
- Dar suporte ao desenvolvimento não linear (muitos ramos ao mesmo tempo)  
- Ser totalmente distribuído  
- Lidar com projetos grandes, como o próprio núcleo do Linux  

Desde sua criação, o Git evoluiu bastante, mantendo essas características e se tornando uma das ferramentas de versionamento mais usadas no mundo.

## Quem criou o git?
 Linus Torvalds, o criador do kernel do Linux

## Quais são os comandos principais do git? 

### `git init`
Inicializa um novo repositório Git.

### `git clone <url>`
Clona um repositório remoto.

### `git status`
Exibe o estado atual dos arquivos no repositório.

### `git add <arquivo>`
Adiciona um arquivo à área de preparação.

### `git add .`
Adiciona todas as modificações à área de preparação.

### `git commit -m "mensagem"`
Registra as alterações no repositório.

### `git pull`
Busca atualizações do repositório remoto.

### `git push`
Envia seus commits para o repositório remoto.

### `git branch`
Mostra todos os ramos.

### `git branch <nome>`
Cria um novo ramo.

### `git checkout <branch>`
Alterna entre ramos.

### `git merge <branch>`
Mescla outro ramo ao ramo atual.

### `git rm <arquivo>`
Remove
