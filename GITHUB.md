# GITHUB

## O que é GitHub ? 
O GitHub é uma plataforma de hospedagem de repositórios Git que permite que pessoas sugiram modificações em projetos e compartilhem conhecimento.
O GitHub permite que os desenvolvedores armazenem, compartilhem e colaborem em projetos de software. Ele é construído em torno do sistema de controle de versão Git, o que facilita o gerenciamento de diferentes versões de um código-fonte. 

## Como funciona o GitHub ?
O GitHub é uma plataforma online que armazena e gerencia projetos que usam Git. Ele funciona como uma rede social de código, permitindo colaboração e controle de versão.

### Repositórios
São “pastas de projeto” na nuvem, contendo arquivos, histórico e branches.

### Commits
São registros das alterações feitas no projeto. Cada commit é como uma foto do estado do código.

### Branches
Ramos usados para criar versões paralelas do projeto sem afetar o código principal.

### Push e Pull
- **Push:** envia suas alterações para o GitHub.  
- **Pull:** baixa as atualizações do GitHub para sua máquina.

### Fork
Cria uma cópia de um repositório de outra pessoa na sua conta, permitindo modificar e contribuir.

### Pull Request
Pedido para mesclar suas alterações ao projeto principal. Usado em colaboração e revisão.

### Issues
Tarefas ou relatos usados para organizar bugs, melhorias e funcionalidades.

### Por que o GitHub é importante?
- Guarda projetos com segurança  
- Organiza o trabalho em equipe  
- Armazena histórico completo  
- Ajuda a criar portfólio  
- Suporta milhões de projetos open source

## Quais os principais concorrentes do GitHub?
### GitLab
Concorrente direto do GitHub. Possui recursos completos de Git e ferramentas avançadas de CI/CD. Muito usado em empresas.

### Bitbucket
Plataforma da Atlassian. Integrado com Jira e oferece repositórios privados gratuitos.

### SourceForge
Antiga plataforma popular para projetos open source. Ainda é usada, mas perdeu espaço para o GitHub.

### AWS CodeCommit
Ferramenta da Amazon para empresas que usam AWS. Suporta repositórios privados e integração com serviços da Amazon.

### Azure DevOps (Repos)
Serviço da Microsoft com suporte a Git, integração com pipelines e foco em equipes grandes e corporativas.
